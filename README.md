# Shadowing Audio Builder (シャドーイング音声作成ツール)

日本語のシャドーイング練習を効率化するために開発された、Python製自動音声処理ツールです。
市販の教材音声を解析し、文と文の間に適切な「空白（ポーズ）」を自動挿入することで、スムーズな反復練習を実現します。

## 🚀 背景 (Background)
語学学習において「シャドーイング」は効果的ですが、通常の教材音声は間隔が短く、初心者がついていくのは困難でした。
そこで、Pythonを用いて音声を加工し、**「聞き取れる・話せる」最適な間隔**を自動生成するツールを自作しました。

## ✨ 主な機能 (Features)
- **Batch Processing**: 複数のユニット（Unit1〜Unit6）を一括で処理可能。
- **Smart Gap Insertion**: 各音声の間に、シャドーイングに最適な空白（2.5秒）を自動挿入。
- **Auto Merging**: 散らばった短い音声ファイルを、ユニットごとに一つのMP3ファイルに結合。

## 🛠 技術スタック (Tech Stack)
- **Language**: Python 3.11
- **Library**: pydub (Audio Manipulation)
- **Engine**: FFmpeg

## 📂 開発履歴 (Dev History)
- `v1`: 基本的な音声結合ロジックの実装
- `v2`: 空白挿入ロジックの追加
- `v3`: 複数ユニットのバッチ処理対応（現在の完成形）
